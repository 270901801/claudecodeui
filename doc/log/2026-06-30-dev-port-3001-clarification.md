# 开发端口规则澄清日志

日期：2026-06-30

## 用户问题

用户问：

> 你为什么要用 5173，你不是只用 3001 就行了吗

## 处理

1. 解释 `5173` 来自 Vite 裸默认端口。
2. 澄清“只用 3001”的准确含义：浏览器访问入口只用 `3001`，后端内部端口使用 `3002`。
3. 更新 `CLAUDE.md`，把快速迭代启动命令改为显式端口：
   - `VITE_PORT=3001`
   - `SERVER_PORT=3002`
4. 更新前序规则文档和端口预检文档，避免后续再把 `5173` 当成本项目目标端口。
5. 新增 `doc/2026-06-30-dev-port-3001-clarification.md` 记录端口解释。
